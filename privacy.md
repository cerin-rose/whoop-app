# Privacy Policy

## Overview
This application uses the WHOOP API to access user fitness data such as sleep, recovery, strain, and workout information.

## Data Access
We only access WHOOP data with the user's explicit permission through OAuth login.

## How Data Is Used
The data is used only to display personal fitness insights inside the application.

## Data Storage
We do not permanently store, sell, or share any user data. Any data used is only for temporary display during the session.

## Third Parties
We do not share user data with any third parties.

## User Control
Users can revoke access at any time from their WHOOP account settings.
